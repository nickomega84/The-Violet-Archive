**The Violet Archive** is a nocturnal, 2000s-inspired web interface. Designed as a hybrid digital sanctuary, it serves as a personal poetry blog, a tribute to the Synth-Goth aesthetic (specifically *The Birthday Massacre*), and an interactive alt/emo diary. 

This project was built from scratch focusing on clean architecture, semantic markup, and zero reliance on external CSS/JS frameworks. It acts as a core piece of my frontend portfolio, demonstrating the ability to craft highly immersive, nostalgic UI/UX experiences for international and remote web environments.

## ⚙️ Technical Features & UI/UX

* **Custom JavaScript Cursor:** A fully functional, custom-built gothic cross cursor injected and tracked via DOM manipulation, featuring hover state scaling and glow effects.
* **Dynamic Multimedia Injection:** To maintain UI integrity and bypass browser autoplay restrictions, the YouTube iframe is dynamically injected into the DOM via JavaScript only when the user clicks the custom HTML/CSS play button.
* **Accordion Navigation System:** A smooth, interconnected accordion menu built with Vanilla JavaScript. It automatically manages state, collapsing inactive tabs to maintain a clean layout.
* **Custom WebKit Scrollbars:** The default browser scrollbars have been overwritten with neon-purple, glowing WebKit trackers to maintain 100% aesthetic immersion.
* **Semantic HTML5 & Pure CSS:** Strict separation of concerns. Zero inline styles. Built with responsive absolute positioning, flexbox, and `@font-face` local font integration.
